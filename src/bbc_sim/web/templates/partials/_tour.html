🏢 bbc-sim Admin へようこそ

このツールは SBCO BACnet B-BC シミュレータ / ゲートウェイ の管理画面です。

実設備なしに BACnet/IP で仮想デバイスを公開し、Building OS / Eclipse Hono / Ditto との結合試験が可能です。

このガイドは初回のみ表示されます。いつでも ヘルプページ で確認できます。

↕️ 北向き / 南向きとは

Building OS
/ Hono / Ditto
↑ BACnet/IP
northbound (常に)
bbc-sim (B-BC)
↓ MQTT / ZeroMQ
southbound (オプション)
フィールドデータ

northbound は常に BACnet/IP です。上流システムは BACnet/IP でデータを取得します。

southbound は MQTT / ZeroMQ / WoT / gRPC — フィールド側のバインディングです（gateway/combined モード時）。

⚙️ 動作モード

simulator
値を内部生成。上流システムのテストに使用。
gateway
南向きから受け取った実データを BACnet/IP に反映。
combined
オブジェクト単位で両方を混在。

現在のモードは画面上部のナビゲーションバーで確認できます。変更にはプロセス再起動が必要です。

📋 主な画面の案内

  • Dashboard — 全体状態の確認（5 秒自動更新）
  • デバイス/オブジェクト — オブジェクト一覧 / 値の書込み / Fault 注入
  • バインディング — 南向き接続状態と最終受信
  • 接続状態 — 北向き BACnet/IP カウンタ（ローカル内省のみ）
  • ログ — プロセス内リングバッファ
  • ポイントリスト — YAML 再読込と差分適用

⚠️ 破壊的操作について

以下の操作は BACnet オブジェクトや実行状態を変更します:

  • 値の書込み — presentValue を変更（上流システムに見える）
  • Fault 注入 — statusFlags / OutOfService を変更
  • ポイントリスト再読込 — オブジェクトの追加・削除・変更

各操作の前に確認ダイアログが表示されます。詳しくは ヘルプ を参照。
